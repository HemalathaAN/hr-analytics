# 👥 HR Analytics — Employee Attrition Analysis

An end-to-end **HR Data Analysis** project using Python, Pandas and Seaborn to analyse employee attrition, salary trends and department performance — helping HR teams make data-driven decisions.

---

## 📌 Project Overview

Employee attrition is one of the biggest challenges for organizations. This project analyses the **IBM HR Analytics Employee Attrition Dataset** to uncover key factors driving employee turnover and provide actionable recommendations to improve retention.

---

## 🎯 Key Business Questions Answered

- 📊 What is the overall attrition rate?
- 🏢 Which department has the highest attrition?
- 💰 How does salary affect employee attrition?
- ⏰ Does overtime impact attrition rate?
- 🎓 Does education level influence attrition?
- 😊 How does job satisfaction relate to attrition?

---

## 🏆 Key Insights

| Insight | Finding |
|---------|---------|
| Attrition Rate | 16% overall attrition |
| Highest Attrition Dept | Sales department (21%) |
| Salary Impact | Low income employees 3x more likely to leave |
| Overtime Effect | Overtime employees have 31% attrition vs 10% non-overtime |
| Job Satisfaction | Low satisfaction employees leave 2x more |
| Age Group | 25-35 age group has highest attrition |

## 🚀 How to Run

### 1. Clone the repository
```bash
git clone https://github.com/HemalathaAN/hr-analytics.git
cd hr-analytics
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Get the Dataset
Download from Kaggle:
- [IBM HR Analytics Dataset](https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset)
- Save as `hr_data.csv` in the project root

### 4. Run the notebook
```bash
jupyter notebook hr_analytics.ipynb
```
